# 200ok

## Setup

* Edit `config.toml` to configure 200ok. 
* Please refer to `config.sample.toml` for format.

### Platform specific

#### Discord

* Put your Webhook URL into `url`.

#### Mastodon

* Put your instance URL (e.g., `https://mastodon.social`) into `url`.
* Put your **access token** into `token`.
